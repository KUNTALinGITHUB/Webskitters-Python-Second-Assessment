# Webskitters-Python-Second-Assessment
Date: 27-04-2026 Full Marks: 100 Duration: 3 Hours

Q1. Employee Work Analysis System
A company maintains daily work logs of its employees. You are asked to create a console-
based Python application to analyze this data.
Requirements:
1. Accept employee details from the user:
• Employee Name
• Department
• Hours Worked
2. Store multiple employee records in a suitable in-memory structure.
3. Implement the following operations:
• Display the total hours worked by employees belonging to each department.
• Display the names of employees who worked more than a specified number
of hours.
• Display all distinct departments available in the records.
4. The application must:
• Run continuously until the user chooses to exit.
• Skip invalid records without terminating execution.
• Stop execution gracefully based on user input.
5. Input rules:
• Employee name should be valid text.
• Hours worked must be numeric.
• Any invalid input must be handled properly.
6. Structure the program using reusable functions.
• At least one function should accept a flexible number of inputs.
• At least one function should use default values.
7. Implement any three of the following features:
• Efficient data transformation logic
• Lazy data processing

• Custom error handling
• Automated function behavior modification
• Safe duplication of complex data
• Pattern-based input validation
• Compact single-line processing logic
8. Output should be formatted clearly and be user-friendly.
